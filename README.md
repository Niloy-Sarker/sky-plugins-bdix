## [Skystream](https://github.com/akashdh11/skystream) Extensions for BDIX Sites.
Currently available:
* CircleFTP
* DhakaFlix
## Usage
1. Open the app and navigate to **Settings**.
2. Select the **Manage Extensions** menu.
3. Click on the **Add Repository** button.
4. Enter the following URL:
```
https://raw.githubusercontent.com/Niloy-Sarker/sky-plugins-bdix/refs/heads/main/repo.json
```
5. Tap **Add**.
6. **Download** the desired plugins.
